Usage
=====

Installation
------------

To use Fashion Finder locally, launch the server through Django

.. code-block:: console

   (.venv) project-fashion-finder/FashionFinderDjango/FashionFinder $ python3 manage.py runserver


Run tests
----------------

To run test ,
you can use the ``Tests(TestCase)`` class:

.. py:function:: def setUp(self)

   Creates a user for testing 
   Returns self.User

   :param self: Self@Tests
   :type: None
   :return: None
   :rtype: None
